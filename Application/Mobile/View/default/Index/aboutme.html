关于我们

  购钢现货网（http://www.gousteel.com），创建于2013年，由上海天予软件科技有限公司和温州海棠文化传媒有限公司共同开发运营，是目前国内资源最多、专业性最强的服务于不锈钢行业的电子商务平台之一。

  购钢现货网，总部设在上海，分部设在中国不锈钢无缝管生产基地——温州，以及浙江松阳、安徽绩溪、福建霞浦等外迁基地为依托，建立起庞大的不锈钢现货数据库，旨为不锈钢生产商、销售商及下游用户提供最精准的不锈钢现货搜索服务。购钢网的推出，打破了传统的找现货、卖现货的瓶颈，精准、方便、快捷，找现货更方便，卖现货更顺畅。我们要打造最专业的不锈钢电子商务平台！

  我们公司不仅提供不锈钢行业电子商务平台，还为企业量身定制：建站服务以及企业形象策划、宣传画册设计等综合服务。我们将秉承“开拓、创新、务实”的企业精神，立足行业，面向全国，打造国内最好的不锈钢电子商务平台！
